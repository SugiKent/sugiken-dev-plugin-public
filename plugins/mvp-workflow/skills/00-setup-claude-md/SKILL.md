# このスキルの位置付け

プロジェクトの `CLAUDE.md` に、AI がコードを扱う際の動作ルール（サンプル 11 か条）を反映する。
一度きりではなく、サンプルが更新されたら再実行して追従させる。

# 手順

1. `references/CLAUDE_SAMPLE.md` を読み込む（このスキルと同じディレクトリの references/ 配下にある）
2. プロジェクトルートの `CLAUDE.md` の有無を確認する
   - **無い場合**: サンプルの内容で新規作成する
   - **ある場合**: ユーザー固有のカスタマイズ（プロジェクト固有のルール・コマンド・構成メモなど）を失わないよう、既存内容とサンプルを統合する。サンプル由来のルール部分はサンプルを正として更新し、それ以外の既存記述は保持する。判断に迷う差分は上書き前にユーザーに確認する
3. `AGENTS.md` に同種のルールが書かれている場合は、`CLAUDE.md` への一本化を提案し、ユーザーの了承を得てから整理する

# 完了報告（必須）

- `CLAUDE.md` を新規作成したか、更新したか（更新の場合は保持したユーザー固有記述の概要）
- 統合・スキップした差分があればその内容
